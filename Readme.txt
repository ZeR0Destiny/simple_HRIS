A simple HRIS

The "Database-Tables/" folder contains sql tables, will need to create a Database called "rfbt". 
Can also create a database name of choice and change the $dbname to the correct database name for "employee-db-manager.php", "file-db-manager.php" and "account-db-manager.php".

The "pages-login.php" is the main page for the user to login, once logged in it will bring the user to "index.php". The user can create new employee, edit existing employee,
add pdf files for each employee. The user can also edit user password if logged as admin.
